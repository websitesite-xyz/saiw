HTML

Hyper Text Mark-up Language

<p> 이렇게 사이에 글자를 넣는거에여 </p>

<a> anchor
: 링크 걸기
: <a href="https://websitesite.xyz/">Websitesite</a>
: Websitesite

<p> paragraph
: 글쓰기
: <p>웹사이트사이트에 오신 것을 환영합니다!</p>
: 웹사이트사이트에 오신 것을 환영합니다!

<br> line break
: 다음 줄로 가기
:<p>웹사이트사이트에 오신 것을<br>환영합니다!</p>
:웹사이트사이트에 오신 것을
 환영합니다!

<ol> ordered list
: 순서가 있는 목록 만들기
: <ol> 목록 제목
        <li>텍스트</li>
        <li>텍스트</li>
        <li>텍스트</li>
  </ol>

    딸꾹질을 멈추는 방법
  1. 숨을 크게 들이쉰다.
  2. 숨을 참을 수 있을 때까지 참는다.
  3. 딸꾹질이 멈췄는지 확인한다.

<ul> unordered list
: 순서가 없는 목록 만들기
: <ul> 목록 제목
        <li>텍스트</li>
        <li>텍스트</li>
        <li>텍스트</li>
  </ul>
